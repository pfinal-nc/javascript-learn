API密钥

获取代理IP

请求地址

GET https://api.proxyhub.com/v1/proxies

请求参数

参数名 类型 必选 说明
country string 否 代理IP所在国家，如：US, CN, JP
protocol string 否 代理协议：http/https/socks5
count integer 否 返回代理数量，默认1个

示例代码

const response = await fetch('https://api.proxyhub.com/v1/proxies', {
    headers: {
        'Authorization': 'Bearer sk_test_your_api_key',
        'Content-Type': 'application/json'
    }
});

const data = await response.json();
console.log(data.proxies);
